# tistory_black_theme
티스토리 Book Club 블랙 테마

## 테마 파일
- `Tistory_black.css`: 기본 블랙 테마
- `Tistory_white.css`: 화이트 테마
- `style.css`: 공통 스타일 및 레이아웃 조정
- `TistoryHtml.html`, `reference.html`: 로컬 미리보기용 템플릿
- `images/`: 아이콘 리소스 (파일명은 소문자+언더스코어 유지)

## 로컬 미리보기
1) 루트에서 `python -m http.server 8000`
2) 브라우저에서 `http://localhost:8000/reference.html` 열기
3) 데스크톱/모바일 뷰에서 레이아웃과 타이포그래피 확인
